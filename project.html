End-to-End CI/CD Automation & Kubernetes Deployment

This project involved designing and implementing a complete DevOps pipeline for a Java-based application hosted on AWS. The pipeline automated build, test, and deployment processes, integrated security scanning using Trivy, and leveraged containerization with Docker and orchestration using Kubernetes.

The project significantly improved deployment speed, reduced manual errors, and enhanced system reliability by implementing Infrastructure as Code (Terraform) and configuration management (Ansible).
